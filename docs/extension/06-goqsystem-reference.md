## 共通機能参考

### 品名印字規則（全配送業者共通 - 優先度高）

4パターン:
1. 商品名の前からN文字を抜き出し
2. 商品名の後ろからN文字を抜き出し
3. 指定キーワード間の文字を抜き出し（例: ■〜★の間）
4. 固定文字を入力（例: 食品、雑貨）

文字数制限: ヤマト25字 / 佐川16字 / 日本郵便25字

### 依頼主切替ロジック

注文者と送付先が異なる場合:
- 依頼主=注文者: 注文者情報を印字
- 依頼主=店舗: 店舗情報を印字 + 「<注文者名>様依頼分」追記オプション

### CSV ファイル名パターン

固定名 / 日付 / 日付時刻（推奨）

### 管理番号印字

印字する（両方）/ しない / 管理番号のみ / 受注番号のみ
