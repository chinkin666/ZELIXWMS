# 日本本土 3PL + 中国来日 FBA 通过型业务 SOP（仓库作业版） / 日本国内 3PL + 中国発日本向け FBA 通過型業務 SOP（倉庫作業版）

## 1. 文档目的 / 文書目的

本文件用于仓库主管、现场收货、检品、贴标、打包、出货人员执行统一作业，并附带可直接复制使用的表单模板。

本書は、倉庫責任者、現場受領担当、検品担当、ラベル担当、梱包担当、出荷担当が統一手順で作業するために使用し、そのまま転用できる帳票テンプレートを含む。

## 2. 适用范围 / 適用範囲

- 日本本土 `3PL` 入出库作业
- 中国发货到日本中转后送 `Amazon.co.jp FBA` 作业
- 退货整备与再发货作业

- 日本国内 `3PL` の入出庫作業
- 中国から日本中継後に `Amazon.co.jp FBA` へ送る作業
- 返品整備および再出荷作業

## 4. 作业前提 / 作業前提条件

以下资料齐全后才可正式作业：

以下資料が揃ってから正式作業を開始する：

- 客户名称 / 顧客名
- 入仓单号 / 入庫指示番号
- `SKU` 清单 / `SKU` 一覧
- 装箱单 / 梱包明細

## 5. 标准作业流程 / 標準作業フロー

### 5.1 收货 / 受領

1. 核对车次、运单号、客户名  
   車番、送り状番号、顧客名を確認する
2. 逐箱点数并记录箱号  
   箱ごとに数量確認し、箱番号を記録する
3. 检查外箱是否有破损、潮湿、变形、二次封箱痕迹  
   外箱の破損、濡れ、変形、再封痕を確認する
4. 异常箱先拍照，贴 `异常待处理` 标识  
   異常箱は先に撮影し、`異常保留` 表示を貼る
5. 收货完成后提交主管复核  
   受領完了後、責任者へ確認依頼を行う

### 5.2 入库登记 / 入庫登録

1. 录入实际到货日期、箱数、重量、体积  
   実到着日、箱数、重量、容積を登録する
2. 对比预报资料与实物  
   事前通知情報と現物を照合する
3. 差异超过约定阈值时，暂停后续作业并报异常  
   差異が許容値を超える場合、後続作業を停止し異常報告する

### 5.3 检品 / 検品

1. 按作业单执行抽检或全检  
   作業指示書に従って抜取検品または全数検品を行う
2. 核对：
   以下を確認する：
- `SKU`
- 条码 / バーコード
- 数量 / 数量
- 外观 / 外観
- 配件 / 付属品
- 包装状态 / 包装状態
3. 发现异常时按类型归类：
   異常発見時は以下に分類する：
- 数量异常 / 数量差異
- 标签异常 / ラベル異常
- 外观不良 / 外観不良
- 包装异常 / 包装異常
- 混装异常 / 混載異常
4. 检品结果必须记录并保留照片  
   検品結果は必ず記録し、写真を保管する

### 5.4 贴标与加工 / ラベル貼付・加工

1. 按指令确认是否贴 `FNSKU`、商品条码、警示贴、箱唛  
   指示に従い、`FNSKU`、商品バーコード、警告ラベル、箱ラベルの要否を確認する
2. 贴标前先核对打印内容  
   貼付前に印字内容を確認する
3. 贴标位置需不遮挡关键商品信息  
   ラベル位置は重要表示を隠さないようにする
4. 完成后由第二人复核  
   完了後、別担当者が再確認する
5. 需要套袋、防漏、防刮、组合的，按要求执行  
   袋入れ、防漏、防傷、セット組みが必要な場合は指示どおりに行う

### 5.5 分箱、合箱、换箱 / 分箱・合箱・箱替え

1. 核对目标箱规与纳品计划  
   目標箱規格と納品プランを確認する
2. 分箱时确保每箱内容明确  
   分箱時は各箱の内容を明確にする
3. 合箱时防止不同计划混装  
   合箱時は異なるプランの混載を防ぐ
4. 换箱后更新箱号、重量、尺寸  
   箱替え後は箱番号、重量、寸法を更新する

### 5.6 FBA 出货 / FBA 出荷

1. 取得最终版纳品计划  
   最終版の納品プランを取得する
2. 核对 `箱数-箱号-箱唛-目的仓` 四项一致  
   `箱数・箱番号・箱ラベル・納品先倉庫` の一致を確認する
3. 打印并贴附配送标签  
   配送ラベルを印刷し貼付する
4. 称重、拍照、封箱  
   計量、撮影、封箱を行う
5. 交运后记录追踪号并回传  
   引渡し後、追跡番号を記録し返却する

### 5.7 退货整备 / 返品整備

1. 收到退货后先区分来源：`FBA退回`、`客户退货`、`拒收返仓`  
   返品受領後、`FBA返送`、`顧客返品`、`受取拒否返送` を区別する
2. 分类：
   分類：
- 可售 / 再販可
- 整备后可售 / 整備後再販可
- 不可售 / 再販不可
3. 按客户指令执行重贴、重包、再发或报废  
   顧客指示に従い、再ラベル、再梱包、再出荷または廃棄を行う

## 6. 关键控制点 / 重要管理ポイント

- 没有作业单，不开工  
  作業指示書がない場合、着手しない
- 没有装箱单，不正式入库  
  梱包明細がない場合、正式入庫しない
- 没有 `FBA` 纳品计划，不得发往 FBA  
  `FBA` 納品プランがない場合、FBA へ出荷しない
- `FNSKU` 贴标后必须二次复核  
  `FNSKU` 貼付後は必ずダブルチェックする
- 异常必须拍照并当日上报  
  異常は必ず撮影し当日中に報告する

## 7. 异常处理规则 / 異常処理ルール

### 7.1 异常级别 / 異常レベル

- `A类`：轻微异常，不影响当日出货  
  `A区分`：軽微異常、当日出荷に影響なし
- `B类`：需要客户确认，但可局部继续作业  
  `B区分`：顧客確認が必要だが、一部作業継続可
- `C类`：影响出货，必须暂停整票  
  `C区分`：出荷に影響し、案件全体を停止する必要あり

## 10. 在库管理 / 在庫管理

### 10.1 适用场景 / 適用場面

本节适用于套餐 A（本土 3PL）长期保管业务。纯 FBA 通过型可跳过本节。

本セクションはプラン A（国内 3PL）の長期保管業務に適用する。純粋な FBA 通過型は本セクションを省略可。

### 10.2 库位管理 / ロケーション管理

1. 所有商品必须指定库位，禁止"落地堆放"
   全商品にロケーションを指定し、「直置き」を禁止する
2. 库位编码规则：`区-列-层-位`（如 `A-03-2-05`）
   ロケーションコード：`エリア-列-段-位置`（例：`A-03-2-05`）
3. 高周转品放近出库口，低周转品放深处
   高回転品は出庫口近くに、低回転品は奥側に配置する
4. FBA 通过型暂存区与长期保管区物理隔离
   FBA 通過型一時保管エリアと長期保管エリアを物理的に分離する

### 10.3 盘点 / 棚卸し

- 月度循环盘点：每月覆盖 `全SKU的20%`，确保季度内全量盘完
  月次循環棚卸：毎月 `全SKUの20%` を対象とし、四半期で全数完了
- 年度全量盘点：每年至少 `1回`
  年次全数棚卸：年 `1回` 以上
- 盘点差异超过 `0.5%` 时，启动原因调查并向客户报告
  棚卸差異が `0.5%` を超える場合、原因調査を開始し顧客に報告する

### 10.4 库龄预警 / 在庫エイジング警告

- `60日` 未出库：通知客户确认处理方案
  `60日` 未出庫：顧客に処理方針の確認を通知する
- `90日` 未出库：开始按超期仓储费率计费
  `90日` 未出庫：長期保管費率での課金を開始する
- `180日` 未出库：要求客户提交书面处理指令（继续保管/退回/废弃）
  `180日` 未出庫：顧客に書面での処理指示を要求する（継続保管/返送/廃棄）

### 10.5 库存状态标识 / 在庫ステータス表示

| 状态 / ステータス | 含义 / 意味 |
|---|---|
| `可用` / `利用可能` | 可正常出库 / 通常出庫可 |
| `待检` / `検品待ち` | 入库后尚未完成检品 / 入庫後検品未完了 |
| `冻结` / `凍結` | 异常待处理、客户指示待确认 / 異常対応中・顧客指示待ち |
| `待退` / `返送待ち` | 等待退回客户或废弃 / 顧客返送または廃棄待ち |

## 12. FBA 移除订单处理 / FBA 返送・所有権放棄対応

### 12.1 场景说明 / 概要

Amazon FBA 会因库龄超期、商品不可售、客户主动申请等原因，将库存退回到指定地址。这与客户退货（Return）是不同的逆向物流场景。

Amazon FBA は在庫エイジング超過、販売不可商品、顧客申請などの理由で、在庫を指定住所へ返送する。これは顧客返品（Return）とは異なる逆物流シナリオである。

### 12.2 接收流程 / 受領フロー

1. 客户提前通知移除订单号（Removal Order ID）与预计到达日
   顧客が事前に返送注文番号（Removal Order ID）と予定到着日を通知する
2. 到货后按移除订单号核对箱数与内容
   到着後、返送注文番号に基づき箱数と内容を照合する
3. 开箱检品，分类为：
   開梱検品し、以下に分類する：
   - 可重新发 FBA / FBA 再納品可
   - 需整备后再发 / 整備後再納品
   - 不可售（损坏/过期/缺件）/ 販売不可（破損/期限切れ/欠品）
4. 分类结果报告客户，等待处理指令
   分類結果を顧客に報告し、処理指示を待つ

### 12.3 后续处理 / 後続処理

| 客户指令 / 顧客指示 | 操作 / 作業 |
|---|---|
| 重新发 FBA | 整备 → 贴标 → 创建新纳品计划 → 发运 / 整備 → ラベル → 新納品プラン → 出荷 |
| 转 FBM 出库 | 入库上架 → 按 FBM 订单出库 / 棚入れ → FBM 注文に従い出庫 |
| 退回中国 | 打包 → 安排国际退运 / 梱包 → 国際返送手配 |
| 废弃 | 拍照留档 → 按规定废弃处理 / 撮影記録 → 規定に従い廃棄 |

### 12.4 费用 / 費用

- 移除订单接收按 `入库费` 标准收取
  返送注文受領は `入庫費` 基準で請求
- 检品、整备、重新贴标按 `增值服务费` 收取
  検品、整備、再ラベルは `付加作業費` で請求
- 废弃处理费另计
  廃棄処理費は別途
