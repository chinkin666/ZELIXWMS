系统概览

ZELIXWMS 是一个以出荷指示、配送数据联动、运单/帐票打印、检品作业和自动处理规则为核心的仓储管理系统。当前代码结构为 Monorepo，包含 frontend、backend、shared 三个主要工作区，数据库为 MongoDB。

  • 前端：Vue 3 + TypeScript + Vite + Pinia + Vue Router。
  • 后端：Node.js + Express + TypeScript + Mongoose。
  • 核心业务：出荷指示、商品主数据、依頼主、配送业者、映射配置、打印模板、帐票模板、自动处理、Yamato B2 连携。
