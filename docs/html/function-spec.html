功能说明书

2. 订单管理功能

  • 支持手工批量创建订单，接口入口为 POST /api/shipment-orders/manual/bulk。
  • 支持条件分页查询、单据详情查看、单笔和批量状态变更。
  • 支持配送回执导入，将 trackingId 及回执字段写回订单。
  • 支持订单批量更新、批量删除、按 ID 批量取回。

3. 商品与主数据功能

  • 商品支持主 SKU、子 SKU、条码、价格、温层、尺寸、图片、库存相关字段。
  • 系统可通过 /api/products/resolve/:sku 解析扫描输入，区分主 SKU 和子 SKU。
  • 依頼主数据用于自动补全发件人信息和 Yamato 发店代码。
  • 配送业者数据包含格式定义和自动化类型，支撑回执导入和运单联动。

4. 映射与导入功能

  • 映射配置负责将外部文件列转换为内部订单字段，适用于 EC 平台或配送数据导入。
  • 转换链支持插件化处理，前后端均存在 transform 类型定义。
  • 用户可维护默认映射、按依頼主/配送业者/配置类型筛选映射。

5. 打印与帐票功能

  • 打印模板支持按画布尺寸、元素、绑定字段生成标签图像或 PDF。
  • 帐票模板支持多列布局、页边距、页眉页脚、条码和日期格式。
  • 前端提供预览、批量打印、Yamato B2 批量 PDF 获取等功能。
